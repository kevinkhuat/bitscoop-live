Ografy

Ografy is a data aggregation and interaction tool that helps you visualize and share your data.

How It Works

Ografy collects the information you generate and re-streams it in a single place.

You can automatically track your activity and easily share cool stats with your friends and the world.

You can easily manage how your data is tracked and published.

Getting Started

To get started with Ografy you'll need to create an account.

Check out the quick start page to get set up.

Next Steps

Ografy was designed to minimize the learning curve.

However, more advanced features are available for users who'd like to spend more time.

If you'd like more detailed control over how your data is visualized, be sure to check out the documentation.

What's Next?

Ografy is continually updated with new features.

If you'd like to check in on the team's status and the planned features, be sure to check out the development blog.
